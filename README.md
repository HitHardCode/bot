# Глаз Бога (Eye of God) - OSINT Бот

Полнофункциональный OSINT (Open Source Intelligence) бот для Telegram, вдохновленный популярными ботами. Собирает и анализирует открытые данные из интернета, социальных сетей, баз данных и других источников.

## Функционал

- **Поиск по людям**: Имена, фамилии, телефоны, email, VK, Telegram
- **Поиск по организациям**: Названия, ИНН, адреса, сайты
- **Поиск по событиям**: Названия, даты, локации, участники
- **Поиск по геолокациям**: Названия, координаты, описания
- **Кросс-референсинг**: Связывание данных между категориями
- **Визуализация**: Карты для геолокаций (Folium)
- **Мониторинг в реальном времени**: Периодические проверки запросов
- **Генерация отчетов**: PDF отчеты с данными
- **Анонимный доступ**: Использование прокси для запросов
- **API интеграция**: REST API для расширения
- **Самообновление базы данных**: Периодическое обновление данных
- **Защита от злоупотреблений**: Rate limiting, аутентификация пользователей
- **Интерфейс на русском**: Команды /поиск, /мониторинг, /отчет

## Этика

Бот использует только открытые источники и не нарушает законы. Все данные собираются этично, без хакерства или взлома.

## Установка

### Локальная установка

1. Клонируйте репозиторий:
   ```
   git clone https://github.com/yourusername/eye-of-god-bot.git
   cd eye-of-god-bot
   ```

2. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```

3. Настройте токен бота в `osint_bot.py`:
   ```python
   TOKEN = "ваш_токен_бота"
   ```

4. Запустите бота:
   ```
   python osint_bot.py
   ```

### Docker

1. Соберите образ:
   ```
   docker build -t eye-of-god-bot .
   ```

2. Запустите контейнер:
   ```
   docker run -e TOKEN="ваш_токен_бота" eye-of-god-bot
   ```

### Docker Compose

1. Запустите с compose:
   ```
   docker-compose up --build
   ```

## Использование

- `/start` - Запуск и меню
- `/поиск <запрос>` - Поиск данных
- `/мониторинг <запрос>` - Начать мониторинг
- `/отчет <запрос>` - Сгенерировать отчет

## Архитектура

- **База данных**: SQLite с SQLAlchemy ORM
- **Бот**: Python Telegram Bot
- **API**: Flask для REST API
- **Визуализация**: Folium для карт, ReportLab для PDF
- **Асинхронность**: AsyncIO для производительности
- **Безопасность**: Rate limiting, аутентификация

## Развертывание

Для продакшена используйте сервер с Python 3.8+, например Heroku, DigitalOcean или VPS.
